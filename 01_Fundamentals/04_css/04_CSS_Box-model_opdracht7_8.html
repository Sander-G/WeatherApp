Opdracht 8

Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height
de totale breedte en hoogte is van het blok van de voorgaande opdracht?

    The alternative CSS box model: Using this model, any width is the width of the visible box on the page,
    therefore the content area width is that width minus the width for the padding and border

    CSS: .box { box-sizing: border-box; }

    of voor alle elementen op de pagina:
    html { box-sizing: border-box; } *, *::before, *::after { box-sizing: inherit; }
