Bij CSS Grid wordt vaak gebruikgemaakt van een speciale lengte-eenheid: “fr”.
Onderzoek wat deze doet en beschrijf het in je eigen woorden.

  • With CSS Grid Layout, we get a new flexible unit: the Fr unit.
    Fr is a fractional unit and 1fr is for 1 part of the available space.
