Opdracht 1.1c

In JavaScript, `const` means that the identifier can't be reassigned.
(Not to be confused with immutable values.
Unlike true immutable datatypes such as those produced by Immutable.
js and Mori, a `const` object can have properties mutated.)


Variables defined with const cannot be Redeclared.
Variables defined with const cannot be Reassigned.
Variables defined with const have Block Scope.
