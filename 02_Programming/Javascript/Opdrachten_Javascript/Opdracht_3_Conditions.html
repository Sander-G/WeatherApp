Opdracht 3.1

1. Wat is het verschil tussen == en ===?

== -> Abstract Equality Comparision : vergelijken van twee variabelen, negeert het datatype

=== -> Strict Equality Comparision : vergelijken van 2 variabelen waarbij datatype ook gelijk moet zijn


2. Wat is het verschil tussen > en >=? En tussen < en <=?

> :groter dan; vs >= :groter dan of gelijk aan.

< :kleiner dan; vs <= :kleiner dan of gelijk aan.
