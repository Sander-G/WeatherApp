    1. Onderzoek de volgende DOM methodes en gebruik ze in een voorbeeld:

  • document.getElementsByClassName:

    • Definitie: Returns an array-like object of all child elements which have all of the given class name(s).

    Voorbeeld: document.getElementsByClassName('red') - alle elementen die de class 'red' hebben

  • document.getElementsByTagName

    • Definitie: Returns an HTMLCollection of elements with the given tag name.

    Voorbeeld: let collection = document.getElementsByTagName("li"); - alle elementen met tag naam 'li'

  • document.querySelector

    • Definitie: Returns the first Element within the document that matches the specified selector,
    or group of selectors. If no matches are found, null is returned.

    Voorbeeld: let el = document.querySelector(".myclass"); - het eerste element in het document met de class "myclass" wordt gegeven

  • document.querySelectorAll

    • Definitie: returns a static (not live) NodeList representing a list of the document's elements that match the specified group of selectors.

    Voorbeeld: let matches = document.querySelectorAll("div.note"); - geeft een lijst van alle <div> elementen met de class "note"

    Opdracht 4.3

    1. Maak een functie randomNumber() en verbindt die aan een button. Zorg dat de functie Math.random() in de console logt.
    Wat merk je op als je deze meerdere keren gebruikt?

    - Wat opvalt is dat telkens een waarde tussen nul en 1 wordt gegeven.

    Math.random() geeft dus een getal 0 >= x < 1 terug - het kan wel 0 zijn, maar nooit 1.
    Als je die laat afronden met Math.round(), zal het vaker afronden naar beneden dan naar boven.
    Als je een willekeurig getal wil tonen, moet je Math.floor() gebruiken op Math.random(). Als je dit zo doet, krijg je altijd 0 terug.

    2. Verander je functie nu dat hij een parameter x gebruikt, en zorg dat hij het volgende in je console logt: Math.floor(Math.random() * x).
    Test nu je functie met verschillende waardes voor x. Valt iets je op?

    de functie geeft nu een random getal tussen 0 en x, waarbij de waarde nooit x zelf bereikt
